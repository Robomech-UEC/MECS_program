# システムの数学モデル

システムとしての制御対象は静的、動的の2つに大別できる。  
静的なシステムとはtにおける入力値のみで出力が決まるシステムで、バネがその一例である。  
動的なシステムとは入出力関係が変数の微分になるものである。
