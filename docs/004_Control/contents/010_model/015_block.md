## システム
制御工学において、制御対象は数式として表されます。そして入力に対してある出力が得られるような数式(つまり制御対象)のことを"システム"と呼びます。  
ここで、システムは必ずしも制御対象のみであるとは限りません。制御器も含めた入出力関係を含むものをまとめてシステムと呼ぶこともあります。  
システムとは特定の式で出力が表される入出力関係なのです。
