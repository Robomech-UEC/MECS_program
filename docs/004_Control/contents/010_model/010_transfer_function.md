## ラプラス変換
### 数式的理解
ラプラス変換とは、関数の表し方を変える方法です。数学的に言うと、時間領域の関数をs領域(複素領域)に変換する方法になります。  
時間領域の関数f(t)に対して、ラプラス変換後の関数F(s)は次のように表されます。  
$$ F(s) = \mathcal{L}\{f(t)\} = \int_{0}^{\infty} f(t) e^{-st}  dt $$
